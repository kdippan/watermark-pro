**Watermark Pro** is a modern, privacy-focused web application that allows photographers and content creators to watermark their images securely. 

Unlike other tools that upload your photos to a remote server, **Watermark Pro processes everything locally in your browser**. This means your high-resolution original photos never leave your device, ensuring 100% privacy and lightning-fast speeds.

## ğŸ› ï¸ Built With

This project relies on powerful client-side libraries to deliver a native-app experience in the browser:

* **[HTML5 Canvas](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)** - The core engine for image manipulation.
* **[Fabric.js](http://fabricjs.com/)** - Advanced canvas object manipulation (layering, drag-drop, scaling).
* **[JSZip](https://stuk.github.io/jszip/)** - Generating ZIP files for batch downloads.
* **[FileSaver.js](https://github.com/eligrey/FileSaver.js/)** - Saving files locally.
* **CSS3 Variables** - For the dynamic Dark/Light theming system.

ğŸš€ Getting Started
Since this is a static web application, you don't need to install Node.js modules or configure a database.
Prerequisites
 * A modern web browser (Chrome, Edge, Firefox, Safari).
Installation
 * Clone the repo
    git clone [https://github.com/kdippan/watermark-pro.git](https://github.com/kdippan/watermark-pro.git)

 * Navigate to the folder
   cd watermark-pro

 * Run the project
   * Simply open index.html in your browser.
   * Note: For better performance and to avoid CORS issues with local images, it is recommended to use a simple local server (like VS Code Live Server).
